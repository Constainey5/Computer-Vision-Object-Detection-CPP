High-performance object detection system implemented in C++ using OpenCV, optimized for real-time applications.

## Getting Started

### Installation
1. Clone the repo
   ```bash
   git clone https://github.com/Constainey5/Computer-Vision-Object-Detection-CPP.git
   ```
2. Build with CMake
   ```bash
   cd Computer-Vision-Object-Detection-CPP
   mkdir build && cd build
   cmake .. && make
   ```
